Welcome to node-common-cpp!
===========================

**node-common-cpp** is a small C++ library providing convinient wrappers around the most often used
elements of the ROS2 API. It's aim is to provide more declartive and self-describing API enabling
quicker and cleaner design of ROS2 utilities.
